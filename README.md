# Book Management System (JavaScript)

A personal library manager built with React + Vite + Tailwind CSS v4 + lucide-react.
Pure JavaScript (no TypeScript). Routing via react-router-dom.

## Setup

```bash
npm install
npm run dev
```

Set your API base in `.env`:

```
VITE_API_BASE_URL=http://localhost:5000
```

The API is expected to expose `/books` REST endpoints (GET, POST, PUT, DELETE).

## Scripts

- `npm run dev` — start dev server
- `npm run build` — production build
- `npm run preview` — preview built app
